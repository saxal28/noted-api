# Noted RESFUL API
> Built With Node, Express, MongoDB. Testing in Mocha.

##GET ROUTES
> **/home** route with author information

> **/notes** lists all notes/note details

> **/users** gets a list of all usernames

##POST ROUTES
> **/notes** add new note
> **/users** add users

##PATCH ROUTES
> **/notes/:id** update note

##DELETE ROUTES
> **/notes/:id** delete note
